The Core MC

Length: Dec 2012 - Oct 2014
Role: Software Engineer

Goal

The Core MC was a minigame network based in Minecraft operating as a F2P SaaS product. Popular minigames include Hot Potato, Control Point, Age of Minecraft, The Core Games, and Blockades. The network was made to give players a variety of games to play with cross game progression, unlocking multiple perks in each game and playing across multiple maps. Additions to the server included pets, global ranks, and a queue system allowing players to queue quickly to join any game.
